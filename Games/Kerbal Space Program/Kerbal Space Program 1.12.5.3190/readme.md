# Archive Server - KSP BakUP

## Overview

This repository serves as a public archive for a specific build of Kerbal Space Program. The files are bundled into a Self-Extracting Archive (SFX) format for long-term storage and data integrity.

## Access and Encryption

The archive is protected by a 32-character encryption key. This measure is implemented to ensure that the server remains a storage repository and not a platform for unauthorized software distribution.

## Legal Disclaimer

This project is an archive for backup  purposes only. 

* The host does not intend to distribute copyrighted material illegally.
